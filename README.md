# BaroCircuit Visualizer

A lightweight, web-based tool for Barotrauma engineers to visualize and export circuit box logic directly from submarine save files. 
This tool shows all circuit boxes on the last active submarine in a save file and their internal circuitry to be exported out in SVG format.

# Features

* .save Support: directly upload your Barotrauma save files and it will list out 

* Game-Accurate Visuals: Renders wires using the Barotrauma colour palette (Orange, Red, Blue, Green, etc.).

* Interactive Viewer: Pan and zoom around logic gates.

* Logic Extraction: Automatically maps internal CircuitBox IDs to their actual component types and properties (Memory values, etc.).

* SVG Export: Download high-quality SVG files of your circuits to share online or include in sub documentation.
    
# How to Use

* Visit the Tool: https://ultraguy24.github.io/baro-circuit-visualizer/

* Upload: Drag and drop your .save file from your Barotrauma local sub-folder into the browser.

* Explore: Select a Circuit Box from the sidebar list.

* Export: Use the Download SVG button to save a snapshot of the schematic.
